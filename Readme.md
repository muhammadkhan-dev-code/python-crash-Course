## üí° Mini Projects

### **[Mini Project 1: Expense Tracker](mini_project1.py)**
A practical expense management system that demonstrates:
- Data structures (dictionaries and lists)
- Function definitions and organization
- User input handling
- Data storage and retrieval
- Expense categorization and tracking
- Running expense calculations

**Features:**
- Add new expenses with date, category, description, and amount
- View all recorded expenses
- Categorize expenses (Food, Transport, Utilities, etc.)
- Track spending over time

### **[Mini Project 2: AI Chatbot Assistant](mini_project2.py)**
An interactive chatbot that showcases:
- Time-based greetings using datetime module
- Response mapping and user interaction
- Keyword matching algorithms
- Conversational AI basics
- Interactive user engagement

**Features:**
- Personalized greetings (Morning/Afternoon/Evening)
- Predefined response library
- Keyword matching for user input
- Motivational responses
- Fallback responses for unknown queries
